# exam_rank_02 practice

This repository contains my practice exercises for the **exam_rank_02** of the 42 curriculum.

## Description

The goal of this repository is to prepare for the **Rank 02 exam** by practicing typical exercises under exam conditions.

These exercises focus on:

- C programming fundamentals  
- String manipulation  
- Memory management  
- Algorithmic thinking under time constraints  

The code is written to match the **exam environment**, where the priority is correctness, efficiency, and speed rather than strict adherence to the 42 Norm.

---

## Structure

Each exercise is implemented in a separate file and follows the format used during the exam:

- No unnecessary dependencies  
- Minimal and concise code  
- Focus on solving the problem as efficiently as possible  

---

## Compilation

All programs are compiled using:

```sh
cc -Wall -Wextra -Werror
```

Example:

```sh
cc exercice.c
./a.out
```

This repository is intended for:

- Reinforcing concepts learned during the cursus
- Practicing problem-solving under pressure
- Improving speed and accuracy for the exam
